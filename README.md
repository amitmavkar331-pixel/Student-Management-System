# Student-Management-System


🎓 Student Management System

📌 Project Description

Student Management System is a full-stack web application built using React and Spring Boot. It helps to manage student data such as adding, updating, deleting, and viewing student records.

---

🚀 Tech Stack

## 🚀 Tech Stack
- Frontend: React.js  
- Backend: Spring Boot  
- Database: MySQL  
- API: REST APIs  
- HTTP Client: Axios
---

✨ Features

- ➕ Add new student
- 📋 View all students
- ✏️ Update student details
- ❌ Delete student
- 🔍 Search student

---

⚙️ How to Run Project

🔹 Backend (Spring Boot)

cd backend
mvn spring-boot:run

🔹 Frontend (React)

cd frontend
npm install
npm start

---

🔗 API Endpoints

- GET /students → Get all students
- GET /students/{id} → Get student by ID
- POST /students → Add new student
- PUT /students/{id} → Update student
- DELETE /students/{id} → Delete student
